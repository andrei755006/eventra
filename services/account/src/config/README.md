# Account Service Configuration

This document describes the environment variables used in the Account service and their purpose.

| Environment Variable | Purpose                                | Default Value |
|----------------------|----------------------------------------|---------------|
| SERVICE_NAME         | Name of the service                    | Account       |
| NODE_ENV             | Application environment                | local         |
| HTTP_PORT            | Port on which the application listens  | 5005          |
| HTTP_HOST            | Hostname where the application runs    | localhost     |
| HTTP_PREFIX          | HTTP prefix for the application routes | /api/account  |
| HTTP_VERSION         | HTTP version of the application        | 1             |
